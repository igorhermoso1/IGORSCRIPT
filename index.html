📦 Variables en JavaScript
Las variables son los contenedores de información de tu programa. Dominarlas es el primer paso.

3 formas de declarar variables:

var nombre = "batman"; // antigua, evitar
let edad = 30; // modificable ✓
const PI = 3.14; // constante, no cambia ✓

// Tipos primitivos
let texto = "Hola Igor"; // string
let num = 42; // number
let activo = true; // boolean
let vacio = null; // null
let sinValor; // undefined

// Template literals (comillas invertidas)
const msg = `Hola ${nombre}, tienes ${edad} años`;

Igor recomienda: usa siempre const por defecto. Solo usa let si sabes que la variable va a cambiar. Nunca uses var en código moderno.

🌐 DOM & Selectores
El DOM es la representación del HTML en memoria. Con JavaScript lo puedes leer y modificar al instante.
// Seleccionar elementos
document.getElementById("miId");
document.querySelector(".miClase"); // primero
document.querySelectorAll(".card"); // todos (NodeList)

// Modificar contenido
elemento.textContent = "Nuevo texto"; // solo texto
elemento.innerHTML = `<b>HTML</b>`; // HTML completo

// Atributos data-*
elemento.dataset.id; // lee data-id
elemento.getAttribute("data-nombre");
elemento.setAttribute("data-id", "99");

// Crear e insertar elementos
const div = document.createElement("div");
padre.appendChild(div);

querySelector acepta CUALQUIER selector CSS. querySelector(".card:first-child"), querySelector("input[type='email']")... lo que CSS puede seleccionar, querySelector también. Mucho más potente que getElementById.

⏳ Async & Fetch API
JavaScript es asíncrono por naturaleza. Dominar fetch y async/await te abre las puertas a cualquier API del mundo.
// Fetch básico con .then()
fetch("https://api.ejemplo.com/datos")
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

async function cargarDatos() {
  try {

  } catch (err) {
    console.error("Error:", err);
  }
}

📚 Arrays & Objetos
Los arrays son listas de datos. Los objetos son estructuras clave-valor. Juntos son la base del JavaScript moderno.
// Arrays
const heroes = ["Batman", "Spawn", "Iron Man"];
heroes.push("Spider-Man"); // añadir al final
heroes.filter(h => h.length > 5); // filtrar
heroes.map(h => h.toUpperCase()); // transformar
heroes.find(h => h === "Spawn"); // encontrar

// Objetos
const personaje = {
  nombre: "Spawn",
  poder: "infernal",
  nivel: 99
};
personaje.nombre; // acceder
personaje["poder"]; // también válido

const url = "https://www.superheroapi.com/api.php/TOKEN/search/spider";
const respuesta = await fetch(url);
const datos = await respuesta.json();
if (datos.response === "success") {
  datos.results.forEach(heroe => pintarHeroe(heroe));
}

El objetivo no es solo “hacer que funcione”, sino aprender a leer results, proteger propiedades como heroe.image?.url y renderizar datos de forma clara.
